# Portfolio Project Notice

This repository is a **portfolio demonstration** of production-grade financial analytics API engineering — view-only, not open source. See [LICENSE](../LICENSE).

## Highlights

- Kotlin · Ktor · Resilience4j · Caffeine · Prometheus · Docker · CI/CD
- Close-of-day returns, alpha, volatility, beta, Sharpe ratio, and rolling correlation
- Alpha Vantage integration with circuit breaker, retry, and analytics caching
